# My Open Source Toolbox

这个仓库包含了几个实用的家庭娱乐工具、数据处理脚本及媒体转换工具。

---

### 🎮 familygame (家庭 HTML 小游戏)
一系列适合家庭互动的轻量化 HTML5 小游戏，无需安装，点击即玩。

* **项目列表：** 女巫的毒药（一款趣味益智小游戏）
* **在线体验：** [点击开始游戏](https://game.madless.club/nwddy.html)

---

### 🏃 strava (数据获取工具)
用于自动获取 [Strava](https://www.strava.com/) 运动平台的骑行、跑步等运动数据。

* **功能：** 支持通过 API 导出活动记录。
* **适用场景：** 运动数据备份、个人仪表盘展示。

---

### 🎵 covert_t_mp3 (媒体格式转换)
一个简单、高效的本地视频转音频工具。

* **主要功能：** 将 `.mp4` 格式视频快速提取并转换为 `.mp3` 音频文件。
* **依赖：** (建议在此标注是否需要 ffmpeg)

---

### 📂 pansou_to_alist (云盘自动化转存)
这是一个强大的自动化工作流工具，实现了从搜索到存储的全自动化。

* **核心链路：** 盘搜搜索 -> 夸克网盘 (quark_auto_save) -> Alist
* **主要特点：**
    * **一键转存：** 搜索结果直接推送到 Alist 管理的存储中。
    * **自动化：** 结合 `quark_auto_save` 解决手动保存的繁琐步骤。
* **依赖组件：**
    * Alist
    * 盘搜接口
    * quark_auto_save

### filetransmission
私有文件传输小程序

* **主要功能：** 可将手机、pc文件传输到当前服务端，再从第三台设备进行下载。
